# Responsive Screenshots


```shell
# install dependencies
npm install
# take a responsive screenshot
node index.js --url="https://material-adaptive.firebaseapp.com/pesto/app/dist.html"
```
![responsive screenshots](screenshots/pesto.png)

### requirements

Chrome 60+, Node 8+
